
############################ 概要 ##########################

MobaSiF は DeNA のモバイルサービスで使われているフレームワークの共通部分を整理して多少の書き直しを行ったものです。素朴で軽いのが特徴です。サービスの要求に合わせてフレームワーク自体を修正していくつくりなので、あまり一般向けではないかもしれませんが、モバイルサイト作りの参考になればと思います。

機能としては、下記のものを提供します。

・URL ディスパッチャ
・端末認証 + ログインまわり
・テンプレート処理 (MTemplate)
・絵文字処理       (Mcode)

その他

・多数の DB ハンドルの管理
・daemon を作る基本モジュール

など。

社内で使っているものと違うところは、

・絵文字の変換タイミングが違う
  - 詳細な説明は省きますが、ezweb の旧機種への対応の関係で
    アプリ側で個別に変換を行っていたのですが、対応世代を絞っているので
    入り口と出口での一律の変換に変更しています。
・FOMA の認証方式が iモードID のみになっている
  - ですが、今のままだと SSL に対応できないので中途半端です。
・セキュリティー対策関連のコードは抜いてある
・主要変数の誤字が修正されている ex) CAREER => CARRIER
・その他

という感じです。また、現状では扱っていないものは、

・iモード SSL 対応
・メール処理

あたりになります。


* 対応端末

・docomo   : FOMA 90x,70x
・au       : WIN
・softbank : 3GC 型


* 動作環境

CentOS     4.x / 5.x (i386)
Apache     1.3.x
MySQL      5.0.x
perl       5.8.x
fastcgi    2.4.x


* 文字コードなど

・docomo   : sjis + xhtml
・au       : sjis + chtml(?)
・softbank : utf8 + chtml

・ソース   : euc
・テンプレ : sjis
・DB       : sjis

（内部処理は sjis）
